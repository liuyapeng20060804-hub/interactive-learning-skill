# 交互式学习 Skill

这是一个 Codex skill，用于把“我要学习 / 我想学 X / 交互式学习”变成一套掌握学习式的一对一个性化课程。

## 安装

在 Codex 里让助手安装这个 GitHub 路径：

```text
https://github.com/liuyapeng20060804-hub/interactive-learning-skill/tree/main/skills/interactive-learning
```

安装后重启 Codex，然后说：

```text
我要学习
```

或：

```text
我要学习 Python
```

## 能做什么

- 为学习主题创建知识图谱和学习路径
- 自动生成 `进度.md` 和每课 Markdown 文件
- 用检查站问题判断掌握程度
- 根据回答生成下一课或补充课
- 始终使用中文教学
